# E-Commerce Project

A simple **Django-based E-commerce API** for managing products, carts, and orders. This project allows users to register, login, browse products, add items to the cart, and make bookings.

---

## Features

- **User Authentication**
  - User registration and login
  - JWT authentication for secure API access
- **Product Management**
  - List products with details like name, description, price, and image
- **Cart System**
  - Add products to cart
  - View cart items
  - Update and remove items from cart
  - Link cart to a specific user
- **Admin Panel**
  - Manage users, products, and carts
  - View cart items along with the user who added them
- **API Testing**
  - Fully testable with Postman or other API clients

---

## Tech Stack

- **Backend:** Python, Django, Django REST Framework  
- **Database:** SQLite (default, can be changed to PostgreSQL)  
- **Authentication:** JWT (JSON Web Token)  
- **Frontend:** HTML/CSS (Optional)
